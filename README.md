# Claude_brain

Durable memory for scheduled Claude Code routines. Start at `CLAUDE.md` for
session startup instructions.

## Routines

- **daily-ai-news** — daily 8am Brasília AI news digest to the private
  `#daily-ai-news` Slack channel (Hacker News, GitHub Trending, Hugging
  Face, X/Twitter). Spec: `daily-ai-news/PLAYBOOK.md`. Feedback log:
  `daily-ai-news/feedback-log.md`.
